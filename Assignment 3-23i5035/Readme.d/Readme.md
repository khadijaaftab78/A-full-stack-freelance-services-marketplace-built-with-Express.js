# TokenBank - Ethereum Token Banking dApp

A decentralized banking application for managing Ethereum and ERC-20 tokens with real-time balance tracking and transaction history.

## Features

- Token Management: Deposit, transfer, and check token balances
- Real-time Updates: Live balance and transaction tracking
- Activity Logging: Complete transaction history with timestamps
- Ethereum Integration: Connect with Ethereum wallets
- User-Friendly Interface: Clean, modern banking interface

## Quick Start

### Prerequisites
- Web3-enabled browser (MetaMask, etc.)
- Ethereum wallet
- Local development server

### Installation
1. Open in VS Code
2. Right-click tokenbank.html
3. Select "Open with Live Server"

### Usage
1. Connect Wallet: Ensure your Ethereum wallet is connected
2. Deposit Funds: Use the "Deposit Funds" button to add tokens
3. Transfer Tokens: Send tokens to any Ethereum address
4. Check Balance: View real-time token and ETH balances
5. Monitor Activity: Track all transactions in the activity log
